Munsell colors matt (spectrofotometer measured)

The reflectance spectra of 1269 matt Munsell color chips.

Measuring equipment: Perkin-Elmer lambda 9 UV/VIS/NIR spectrofotometer.

Wavelength interval: 380 nm - 800 nm.

Wavelength resolution: 1 nm.

File formats:  The available formats are ASCII and MATLAB(tm).

The file munsell380_800_1.asc.gz contains the ASCII format spectra in one file.
Every single color contains values from 421 channels from 380 nm to 800 nm
with one nanometer steps and there are 1269 different colors measured, so in the
file there are 421*1269 = 534 249 values totally. For example the first color 2.5
R 9/2 contains values from 1 to 421 and second color 2.5 R 8/2 values from 422 to
842 and so on. The index of the ASCII spectra is at the end of this README.txt file. 

The file munsell380_800_1.mat.gz is a MATLAB(tm) file which includes 3 matrices
named:

   munsell      421x1269 matrix where each column is one 421 component spectrum.

   S            1269x12 matrix where 1st row is the label of 1st spectrum in
                munsell matrix, 2nd row is the label of 2nd spectrum and so on.

   C            16x1269 matrix where 1st column consists of the color coordinates
                of the 1st spectrum in munsell matrix, the 2nd column consists of
                the color coordinates of the 2nd spectrum and so on. The color 
                coordinates are calculated by using D65 light source and they are
                stored in the column vectors in following order:

                C(1)    x
                C(2)    y
                C(3)    z
                C(4)    X
                C(5)    Y
                C(6)    Z
                C(7)    R
                C(8)    G
                C(9)    B
                C(10)   L*
                C(11)   a*
                C(12)   b*
                C(13)   u
                C(14)   v
                C(15)   u*
                C(16)   v*
